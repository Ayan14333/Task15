# Task15 -(Array & Function Operations in JavaScript)

## Project Description
This project demonstrates basic array operations using JavaScript functions. It covers essential concepts like finding the maximum value, calculating the sum, and counting odd numbers in an array.

## Features
- Find the maximum number in an array
- Calculate the sum of all elements
- Count the number of odd numbers
- Uses different types of functions:
 - Normal Function
 - Arrow Function
 - Anonymous Function


## Concepts Used
- Arrays
- Loops (for, for...of)
- Conditional Statements (if)
- Functions in JavaScript


## How to Run
- Open any code editor (VS Code recommended)
- Copy the code into a .js file

- Run using:

- node filename.js

- OR open browser console and paste the code
